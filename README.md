# BrightTV_Project

This is a project covers the analyses of Bright TV users and viewership. The purpose of the project was to assist the CVM (Customer Value Management) to meet this years objective, which is to increase viewership. The project utilised previous data which was comprised of user Id's, channels viewed, province, race, age, gender and viewing times.

The Methodology and tools used for the project are as follows:

Miro for project planning and brainstorming
Canva to create a project gantt chart to ensure the project met the submission deadline
SQL queries in databricks, to analyse the data and create columns from which meaningful insights can be drawn, Improtant instructions from the CEO was the UTC time had to be converted to SAST and the string date column to be split into date and time.
Excel for data analysis and visualition (the use of bar-graphs, doughnut charts, pie charts and line graphs to understand the data)
Canva to create a presentation for the CEO
The project aimed at sharing findings and recommendations, to aid in increasing viewership.
